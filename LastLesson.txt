3. Các công nghệ sử dụng
3.1. Frontend - ReactJS
- state, props, ant-design
- routing: Navigate, Link, get search params trên URL
- viết API lấy data từ BE
3.2. Backend - Nodejs - Express
- Luồng xử lý: app.js (app.use(...)) -> các router (folder routes) -> các controllers (thao tác đến DB)
- Lưu thông tin người dùng đã đang nhập bằng passport, xác minh các request đến BE
- batch trong firestore
3.3. firestore
- Collections, documents
- Cấu trúc các collections: users, products, cartProducts, carts liên quan với nhau ntn
3.4. Git - Sourcetree: quản lý version code, tách các tính năng thành các nhánh cho các thành viên phát triển
